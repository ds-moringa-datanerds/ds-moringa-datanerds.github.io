<h2>What to Expect</h2>

<p>Instead of learning concepts in isolation, you'll build a <strong>real data science project</strong> step by step across all 6 modules. Each module adds a new layer:</p>

<div class="card-grid">
  <div class="card">
    <h3>🐍 Module 1</h3>
    <p>Set up the project, write Python scripts to collect and structure raw data.</p>
  </div>
  <div class="card">
    <h3>📊 Module 2</h3>
    <p>Explore the data, create visualizations, query with SQL, clean and prepare datasets.</p>
  </div>
  <div class="card">
    <h3>☁️ Module 3</h3>
    <p>Scale to larger datasets, apply statistical tests, build regression models, create dashboards.</p>
  </div>
  <div class="card">
    <h3>🤖 Module 4</h3>
    <p>Apply machine learning — classification, clustering, and model evaluation.</p>
  </div>
  <div class="card">
    <h3>🧠 Module 5</h3>
    <p>Add NLP, time series, or neural network components to your project.</p>
  </div>
  <div class="card">
    <h3>🏗️ Module 6</h3>
    <p>Polish everything into 3 portfolio-ready capstone deliverables.</p>
  </div>
</div>
