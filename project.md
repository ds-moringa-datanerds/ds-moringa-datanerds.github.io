<h2>Module 1 — Python Toolkit: Personal Task Manager</h2>

<p>In Module 1, you'll build a <strong>Personal Task Manager</strong> from scratch using only Python. It starts simple and grows more capable as you learn new concepts. By the end of the module, you'll have a working command-line tool that you built piece by piece.</p>

<p>Whether you've never written a line of code or you've been tinkering with Python before, this project meets you where you are — beginners get to see their first real program come to life, and those with experience get to practice writing clean, structured code.</p>

<p>By the end of Module 1, your task manager can create tasks, set priorities, mark them complete, search by keyword, and save everything to a file. You built it. You understand every line. That's the foundation everything else builds on.</p>

<p>As you progress through the programme, the project evolves with you. Each module introduces new tools and techniques that extend what you've already built — from data analysis and visualization to machine learning and beyond. Details will be shared as we reach each module.</p>
